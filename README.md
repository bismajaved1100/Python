# Python Mini-Projects
Simple, interactive Python programs designed to showcase fundamental programming logic, file manipulation, and standard libraries.

---

## Included Projects

### 1. Calculator with History
A terminal-based calculator that performs basic arithmetic operations while maintaining a running log of your calculations.
* **Key Features:** Uses **File Handling** to persistently save your calculation history to a text file, allowing you to review past results even after closing the program.

### 2. Words Guessing Game
An interactive text puzzle game where the player tries to guess a hidden word within a set number of attempts.
* **Key Features:** Implements Python's built-in **`random` module** to select a fresh secret word automatically every time you start a new game.

---

## Concepts Covered
* Control flow (loops, conditionals, and functions)
* Persistent File I/O (Reading and writing files)
* Working with standard library modules (`random`)
* Input validation and user-friendly console interfaces
